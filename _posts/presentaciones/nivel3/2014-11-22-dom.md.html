--- categories: [presentaciones, nivel3] title: "(nivel 3) Sesión 1: programando la web con el DOM" layout: slideshow --- class: center, middle, inverse # DOM: programando la web --- ## Qué es el DOM? * **D**ocument **O**bject **M**odel: el _documento_ tratado como un _objeto_ * Sí, un objeto de javascript * Una interfaz estándar que permite presentar, inspeccionar y manipular documentos HTML * JS en los browsers trae una implementación del estándar. --- class: center, middle # HTML --- ## HTML: términos útiles * _Hypertex Markup Language_: una forma de especificar la **estructura** de un documento. * **Elements (Elementos)**: componentes de una página * **Tags (etiquetas)**: marcan el inicio/fin de un elemento. Algunos elementos pueden *contener* otros elementos. * **Attributes (atributos)**: información adicional acerca de un elemento. * **Documents (documentos)**: una estructura de html con un elemento principal: `html`, una cabeza (`head`) y un cuerpo (`body`) * El elemento `head` tiene información *acerca* del documento, `body` es el documento en sí. * Componentes especiales: doctype y comentarios. * **Doctype**: declara la versión de html. * Vamos a estar hablando de [HTML 5](http://diveintohtml5.info/). * Todos los ejemplos pueden ser copiados a [thimble](https://thimble.webmaker.org/) --- ## HTML: un documento de ejemplo [Click aquí para verlo en el browser](http://www.rizomas.org/javascript/codigo/ejemplo.html) ```html Demo de HTML
